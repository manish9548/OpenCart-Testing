# 🛒 OpenCart E-commerce: End-to-End Manual Testing Project

## 📌 Project Overview
This repository showcases a professional implementation of the **Software Testing Life Cycle (STLC)** on the OpenCart e-commerce platform. From analyzing requirements to documenting critical bugs and final reporting, this project demonstrates industry-standard manual testing practices.

---

## 📁 Repository Structure
- **Documents/**: Contains the Master Excel sheet (Test Plan, 40+ Test Cases, RTM, Bug Report, Summary).
- **Screenshots/**: Categorized visual evidence of test execution and identified defects.

---

## 🛠️ Testing Artifacts Included
1. **Test Plan**: Scope, strategy, and environment setup.
2. **Test Cases (40 Scenarios)**: Covering Login, Search, Cart, Checkout, and Admin modules.
3. **Requirement Traceability Matrix (RTM)**: Mapping 100% test coverage.
4. **Bug Report**: Detailed documentation of 8+ defects with severity and priority.
5. **Test Summary Report**: Final execution metrics and release recommendations.

---

## 📊 Execution Metrics
- **Total Test Cases:** 40
- **Passed:** 32 (80%)
- **Failed:** 08 (20%)
- **Critical/High Severity Bugs:** 4
